Web Development Part 2

Language

  1. HTML stands for Hyper Text Markup Language
  2. HTML is the standard markup language for creating Web pages
  3. HTML describes the structure of a Web page
  4. HTML consists of a series of elements
  5. HTML elements tell the browser how to display the content
  6. HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc


HTML is a markup language

What is Markup Language?

A way to communicate which way you want to display your content using a code language.

Why is it important?

It is implemented in most modern browsers and still expanding. It is also the current W3C (World wide web consortium) recommendation for web development.

Validating a website through W3C is an important step which can dramatically help to improve and ensure your website's quality.

Elements and Attributes
HTML elements explained:

  1. The element name within angled brackets
  2. HTML elements can also contain attributes

Attributes define additional characteristics or properties of the element such as width and height of an image.

Semantic Markup

Focus on the structure of the document, not how it looks
Advantages:
  • Maintainability
  • Performance
  • Accessibility
  • Search Engine Optimization

Structure

HTML5 does not require the use of the html, head, and body elements (but developers tend to keep using these tags)

  • Html contains all the other HTML elements in the document
  • Head contains descriptive elements about the document - ie. title, style sheets & Javascript files
  • Body contains content to be displayed by the browser
  • Meta element declares that the character enoding for the doc.
  • External CSS style sheet file with LINK that is used with the document
